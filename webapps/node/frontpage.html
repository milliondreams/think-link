Skeptical Readers

  1. Install the Dispute Finder Extension
  2. Dispute Finder highlights disputed claims on pages you read on the web
  3. Click on a claim to see web articles that support other points of view

Activists

  1. Install the Dispute Finder Extension
  2. Tell Dispute Finder about a claim on the web that you disagree with
  3. Train Dispute Finder to find pages on the web that make this claim
  4. Give Dispute Finder arguments for other points of view
  5. Skeptical Readers will now be told that this claim is disputed when they read a page on the web that makes this claim
